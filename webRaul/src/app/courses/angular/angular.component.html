Angular

Primeros pasos con Angular

Instalar Angular

  • A través de npm: > npm install -g @angular/cli
  • Comprobamos la instalación: > ng --version

Iniciar el proyecto

  • Crear proyecto: > ng new nombreProyecto
  • Para levantar el servidor en el puerto http://localhost:4200 : > ng serve, tambien puedes incluir el comando: -o y se abrira el navegador por defecto.

Comandos para crear elementos

  • > ng generate tipoDeElemento nombre
  • Tipos de elementos: component, directive, route, pipe, service, class, interface y enum.
  • Ejemplos:
    • > ng generate component inicio
    • > ng g c inicio, ejemplo anterior simplificado.
    • > ng generate interface interfaces/persona, se puede crear una ruta para mayor organizacion. En este caso se creara una carpeta interfaces con el interface persona

Build del proyecto

  • > ng build
  • > Si se necesita empaquetar el codigo añadir el sufijo --prod

Componentes

Los componentes son vistas que se pueden reutilizar y organizar en función de las necesidades de nuestro proyecto

Generar componentes

Aqui generamos un componente "inicio", se crean tres archivos del componente .html, .css y .ts

En el archivo .htm se encuenta la vista, la estructura es igual a la de html.

En el archivo .css se definen los estilos.

En el .ts se ejecuta Javascript

  • import: Importaciones de archivos necesarios
  • @component: Se encuentra el selector que hace referencia al componente, el template y los estilos, que al ser un array podemos crear todos archivos .css que necesitemos
  • export: Es el ambito de nuestra clase
  • constructor: dentro de los parentesis( inyectables ), dentro del ambito; inicialización de variables
  • ngOnInit: Aquello que se ejecutara al iniciar el compononte.

Anidar componentes

Existe la posibilidad de anidar componentes simplemente añadiendo el selector de un componente en el .html de otro componente

Anteriormente tienes que importar y declarar dichos componentes en el app.module.ts, suelen importarse automaticamente si se general por comando por el terminal
