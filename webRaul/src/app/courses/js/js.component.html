JavaScript

Introducion

Javascript es un lenguaje debilmente tipado que se ejecuta sincronamente, se refiere a que los datos pueden ser modificados independiente del tipo de dato y tiene un flujo de ejecucion sincrono.

Para ejecutarlo solo se necesita un navegador y escribir codigo en su consola o un archivo html cargar el archivo js.

Como se carga JavaScript

Creamos un archivo .js en nuestra carpeta del proyecto a la altura del archivo html, o en una subcarpeta pero teniendo en cuanta la ruta para luego poder acceder a dicho archivo

  • Dentro del .html y al final del head con una etiqueta script y crear todo el codigo js dentro (no recomendable porque ensucia el código html).
  • Dentro y al final del head con una etiqueta script añadir el atributo src y cargar la ruta del archivo JS creado anteriormente 'src="archivoJs" (no recomendable porque el script hace referencia al Dom y el Html carga despues) '
  • Como las dos opciones anteriores pero al final del body , ya que el html necesita cargar antes el script. Siendo la mejor forma crear un archivo JS y cargarlo a traves de etiqueta script src="archivo.js".

Despues de vinvular en el .html el archivo .js empecemos a ejecutar codido en el .js

Variables y constantes

Son un tipo de contenedor donde podemos guardar datos y en algunos casos modificarlos

Constantes

Se usa la palabra reservada const para la declaracion

Dato constante, no se puede reasignar pero si se podran modificar sus valores con ciertos momentos y con sus metodos.

  • const number = 5;

Las constantes puede asignar cualquier tipo de dato; sting, objeto, array, boolean, etc

Variables

var. Como su nombre indica es un contenedor de datos variables. Al no ser un lenguaje fuertemente tipado el tipo de dato puede variar.

  • var number = 5;
  • console.log(number) // 5
  • number = "Cambio a texto"
  • console.log(number) // "Cambio a texto"

En las variables se utiliza la palabra reservada let como var dependiendo el hambito

Ámbito

Dependiendo de donde se declaren las variables y constantes tienen un hambito de funcion o de bloque

Globales

Todas las constantes y variables que estan en la raiz del codigo son accesibles desde cualquiel punto del codigo

De función

Todas las que esten declaradas dentro de una funcion no se pueden acceder a ellas desde fuera

De bloque

En este ejemplo se observa como dentro de una primera funcion se declaran tres variables var dentro de un bloque, acto seguido fuera del bloque de declaracion hay un console.log que hacen referencia a esas variables pudiendo acceder a su valor

En la segunda funcion que es exactamente la misma pero declarando const y let vemos que el console.log no puede acceder a estas

El DOM

Document Object Model

Nodos

Las paginas HTML pueden manipular estos nodos;

  • Document, es el nodo raiz de donde derivan los demás nodos
  • Element, representa a las etiquetas HTML.
  • Attr, representa a cada uno de lños atributos.
  • Text, representa al texto de una etiqueta HTML.
  • Comment, representa a los comentarios incluidos en el HTML.

Obtener Nodos

Obtener uno o varios nodos segun, la funcion y el parametro que se le pase;

  • getElementByTagName ( ), obtiene tos los selectores con la etiqueta html indicada;
  • getElementByName( ), igual que la anterior por esta vez a través de atributo name con valor (x)
  • getElemnetById( ), nodo con dicho id en el html
  • getElemnetByClassName( ), obtiene todos los elementos con esa clase
  • querySelector( ) , en este caso tienes que añadir los selectores como si se tratase de CSS, solo obtendra el primer elemento que contenga el selector
  • querySelectorAll( ), todos los selectores que coincidan con el selector

Añadir nodos al DOM

Son varios pasos;

  1. var parrafo = document.createElement("p")=> creas el elemento
  2. var texto = document.createTextNode("Soy un parrafo") => creas el texto
  3. parrafo.appendChild(texto) => añades el texto al elemento
  4. document.body.appendChild(parrafo) => añades el elemento creado al elemento del dom

appendChild es una funcion que inserta como hijo el parametro indicado entre los corchetes

El paso 3 se puede omitir de la siguiente manera;

document.body.appendChild(parrafo.appendChild(texto))
