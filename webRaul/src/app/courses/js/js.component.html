El DOM

Document Object Model

Nodos

Las paginas HTML pueden manipular estos nodos;

  • Document, es el nodo raiz de donde derivan los demás nodos
  • Elemet, representa a las etiquetas HTML.
  • Attr, representa a cada uno de lños atributos.
  • Text, representa al texto de una etiqueta HTML.
  • Comment, representa a los comentarios incluidos en el HTML.

Obtener Nodos

Obtener uno o varios nodos segun, la funcion y el parametro que se le pase

  • getElementByTagName, obtiene tos los selectores con el selector "p"; var parrafos = getElementByTagName("p")
    • Devuelve un array con todos los nodos, pero un array HTML Collection, que para acceder a uno en cocreto hay que indicar la posicion entre corchetes; var segundoP = parrafos[1]
    • Se podrian recorrer todos con un bucle for
    • Para adentrar en algunas de sus propiedades se le añade un punto y la propiedad; var segundoP = parrafo[1].textContent
  • getElementByName(), igual que la anterior por esta vez a través de atributo name con valor (x)
  • getElemnetById(), nodo con dicho id en el html
  • querySelector() , en este caso tienes que añadir los selectores como si se tratase de CSS, solo obtendra el primer elemento que contenga el selector

    • Ejemplo: var parrafo = document.querySelector("div.banner ul.list li:first-child)

  • querySelectorAll(), todos los selectores que coincidan co el selector

Añadir nodos al DOM

Son varios pasos;

  1. var parrafo = document.createElement("p")
  2. var texto = document.createTextNode("Soy un parrafo")
  3. parrafo.appendChild(texto)
  4. document.body.appendChild(parrafo)

appendChild es una funcion que inserta como hijo el parametro indicado entre los corchetes

El paso 3 se puede omitir de la siguiente manera;

document.body.appendChild(parrafo.appendChild(texto))

Variables y constantes

Son un tipo de contenedor donde podemos guardar datos y en algunos casos modificarlos

Constantes

Se usa la palabra reservada const

Dato constante, no se puede reasignar pero si se podran modificar sus valores con ciertos metodos.

  • const number = 5;
  • console.log(number) // 5
  • number = "Cambio a texto" //Error se puede asignar otro valor a una constate

Las constantes puede asignar cualquier tipo de dato; sting, objeto, array, boolean, etc

Variables

var. Como su nombre indica es un contenedor de datos variables. Al no ser un lenguaje fuertemente tipado el tipo de dato puede variar.

  • var number = 5;
  • console.log(number) // 5
  • number = "Cambio a texto"
  • console.log(number) // "Cambio a texto"

En las variables se utiliza la palabra reservada let como var dependiendo el hambito

Ámbito

Dependiendo de donde se declaren las variables y constantes tienen un hambito de funcion o de bloque

Globales

Todas las constantes y variables que estan en la raiz del codigo son accesibles desde cualquiel punto del codigo

De función

Todas las que esten declaradas dentro de una funcion no se pueden acceder a ellas desde fuera

De bloque

En este ejemplo se observa como dentro de una primera funcion se declaran tres variables var dentro de un bloque, acto seguido fuera del bloque de declaracion hay un console.log que hacen referencia a esas variables pudiendo acceder a su valor

En la segunda funcion que es exactamente la misma pero declarando const y let vemos que el console.log no puede acceder a estas
